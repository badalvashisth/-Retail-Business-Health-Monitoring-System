# 🛒 Retail Business Health Monitoring System

## 📌 Project Overview

The **Retail Business Health Monitoring System** is an end-to-end Data Analytics project developed using **Python, MySQL, SQL, and Power BI**. The project analyzes a **large-scale retail dataset containing over 500,000 transaction records** to uncover business insights, monitor sales performance, and support data-driven decision-making.

The workflow covers the complete analytics lifecycle, including data cleaning, feature engineering, SQL-based business analysis, and interactive dashboard development.

---

# 🎯 Business Problem

Retail businesses generate massive amounts of transactional data every day. Without proper analysis, identifying revenue trends, top-performing products, seasonal demand, and country-wise sales performance becomes difficult.

This project provides an interactive Business Intelligence solution that helps management monitor business performance and make informed decisions.

---

# 📊 Dataset Information

* Dataset: Online Retail II

* Data Type: E-commerce Sales Transactions
* Fields Included:

  * Quantity

# 🧹 Data Preprocessing

The raw dataset was cleaned and transformed using Python.

Performed operations include:

* Removed duplicate records
* Handled missing values
* Converted data types
* Created Revenue column
* Extracted Date
* Extracted Time
* Created Month column
* Created Month Name
* Created Weekday column
* Created sorting columns for Month and Weekday
* Prepared dataset for SQL and Power BI
